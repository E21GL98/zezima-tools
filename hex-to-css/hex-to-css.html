HEX TO FILTER CSS by SoN

L'obiettivo è quello di poter creare fogli di stile personalizzati e consentire la colorazione delle icone SVG.

Affinché questo codice funzioni bene, il colore iniziale deve essere nero. Se il tuo set di icone non è nero, puoi preporre "brightness(0) saturate(100%)" alla proprietà del filtro che prima trasformerà l'icona impostata in nero.
